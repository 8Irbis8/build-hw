# Build HW Project
Проект с настройкой сборки через esbuild и SWC для React.js приложений и библиотек.

## Основные скрипты сборки

```javascript
// Dev сборка приложения
npm run dev:app

// Production сборка приложения
npm run build:app

// Сборка библиотеки через esbuild
npm run build:lib

// Сборка библиотеки через esbuild + SWC трансформатор
npm run build:lib:swc

```
